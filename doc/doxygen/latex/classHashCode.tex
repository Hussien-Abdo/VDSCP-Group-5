\doxysubsection{Detailed Description}
\mbox{\hyperlink{classHashCode}{Hash\+Code}} class is used to represent the Unique tuple which stores the data of each node in the ROBDD. Data include\+:
\begin{DoxyItemize}
\item Human readable label
\item High successor of the node
\item Low successor of the node
\item Top\+\_\+var of the node 
\end{DoxyItemize}
