\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classClassProject_1_1Manager}{Manager}} class is the main class in this package. It implements \mbox{\hyperlink{classClassProject_1_1ManagerInterface}{Manager\+Interface}}. It contains all BDD package functions. In addition it implements the ROBDD as an unordered\+\_\+map. The order of the variables -\/priority-\/ is the insertion order.
